# Backend Core

A minimal backend system built with Python and SQL to rebuild fundamentals
before transitioning to Django.

## Purpose
This project exists to rebuild Python fundamentals through real backend use cases.
The focus is on database interaction, transactions, error handling, and clean structure.

## Tech Stack
- Python 3.x
- SQL (SQLite / MySQL)
- Git

## Project Structure
src/
 ├── main.py        # Entry point
 ├── config.py      # Environment & config
 ├── db/            # Database layer
 └── services/      # Business logic

## How to Run
1. Create and activate a virtual environment
2. Install dependencies
3. Create a `.env` file using `.env.example`
4. Run `python src/main.py`

## Learning Goals
- Use Python as a backend tool
- Handle database transactions safely
- Understand what Django abstracts internally
